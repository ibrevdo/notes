VOIP

RTP

A protocol for delivering audio and video over networks
Typically run over UDP
Used in conjunction with RTCP (RTP control protocol) which delas with QoS of the media streaming
Used in conjuction with SIP (Session Initiation Protocol) which establishes connections across the network

RTP specification (RFC3550) https://tools.ietf.org/html/rfc3550 https://en.wikipedia.org/wiki/Real-time_Transport_Protocol

SIP

SIP specification (RFC3261) https://tools.ietf.org/html/rfc3261 https://en.wikipedia.org/wiki/Session_Initiation_Protocol

SDP

SDP carried as payload in SIP messages. SDP specification (RFC4566) https://tools.ietf.org/html/rfc4566 https://en.wikipedia.org/wiki/Session_Description_Protocol
