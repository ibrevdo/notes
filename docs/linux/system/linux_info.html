Logs

The following files contain Linux logs. These files contain info from last boot

  • /var/log/kern.log captures only the kernel's messages of any loglevel
  • /var/log/messages non-critical and non-debug messages. This log should be considered the "general system activity" log.
  • /var/log/syslog logs everything, except auth related messages
