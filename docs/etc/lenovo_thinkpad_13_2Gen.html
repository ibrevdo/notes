Boot loader : grub2

  • to switch to linux:

    • edit /etc/default/grub: GRUB_DEFAULT=0
    • grub-mkconfig -o /boot/grub/grub.cfg
    • reboot

  • to switch to windows:

    • edit /etc/default/grub: GRUB_DEFAULT=2
    • grub-mkconfig -o /boot/grub/grub.cfg
    • reboot
